

Install Virtualenv


~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
python3 -m pip install virtualenv
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Creating Virtualenv


~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
python3 -m venv <any name for env>
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Activate Virtualenv


~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
bash/zsh: source env/bin/activate
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~



To Install Requirments File


~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
pip3 install -r requirements.txt
pip install -r requirements.txt
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
